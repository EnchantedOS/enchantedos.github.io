## What is Enchanted? {{ anchor: true }}

Enchanted brings modern development practices to Minecraft server development, combining the power of TypeScript, Deno, and real-time WebSocket communication. {{ className: 'lead' }}

The framework consists of three main components:

- **Denorite Server**: A high-performance WebSocket server for real-time communication
- **Fabric Plugin**: Seamless Minecraft server integration through a modern mod
- **Web Operating System**: Intuitive Vue.js-based interface for server management

## Key Features {{ anchor: true }}

### Modern Development

Build your server enhancement modules using TypeScript with hot-reloading support for rapid development. {{ className: 'lead' }}

### Real-Time Communication

Experience instant interactions with your Minecraft server through our WebSocket-based event system. {{ className: 'lead' }}

### Modular Architecture

Create encapsulated functionality using our decorator-based module system and event-driven design. {{ className: 'lead' }}

## Quick Example {{ anchor: true }}

```typescript {{ title: 'welcome.ts' }}
@Module({
  name: 'WelcomeModule',
  version: '1.0.0'
})
export class WelcomeModule {
  @Event('player_joined')
  async onPlayerJoin({ params, api }: ScriptContext) {
    const { playerName } = params;

    await api.tellraw(playerName, JSON.stringify({
      text: "Welcome to the server!",
      color: "gold",
      bold: true
    }));
  }
}
```
